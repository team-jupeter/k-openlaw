SHA-256 이중 재해싱 — 7단계

  1. ①최하위 공통 계층 식별
  2. ②문서 내용 SHA-256 해싱 → 문서 해시값(400) 생성
  3. ③현재 타임스탬프 생성
  4. ④해시값 ‖ 타임스탬프 연결 후 1차 재해싱
  5. ⑤1차 결과에 대해 2차 재해싱
  6. ⑥결과값 mod 100 으로 범위 변환
  7. ⑦사전 확률 분포 테이블과 비교 → 기록 계층 선택
실측: 1,024B 문서 10,000회 반복 → 평균 0.0127ms (P95: 0.0186ms)

이중 서명 메커니즘 — 사용자 주권 + 데이터 무결성 동시 보장

모든 사용자의 해시 체인 갱신은 두 개의 디지털 서명을 동시에 요구합니다. 노드가 사용자 동의 없이 체인을 변경할 수도, 사용자가 노드 검증 없이 체인을 조작할 수도 없습니다.

서명 1 — 사용자 디지털 서명
사용자 자신의 개인 키로 서명. 해시 체인 갱신에 대한 명시적 동의를 의미하며, 노드가 임의로 체인을 변경하는 것을 방지합니다.
서명 2 — 읍면동 노드 디지털 서명
사용자가 소속된 읍면동 계층(100) 노드의 서명. 거래 검증 완료를 인증하며, 사용자가 노드 없이 체인을 조작하는 것을 방지합니다.

이중 지출 방지 메커니즘 — 5가지 방법 (조합 적용)

1
연결 상태 기반 잠금 — 단말기가 한 노드와 연결 중일 때 다른 노드로 새 거래 전송 불가. 연결 해제 = 해시 체인 갱신 완료 신호.
2
읍면동 노드 잠금 메시지 — 해시 접수 즉시 소속 읍면동 노드에 계정 잠금 메시지 전송. 잠금 확인 후에만 해시 체인 융합 개시.
3
노드 해시 체인 제어권 — 해시 전송 시점~갱신 완료까지 해당 노드가 사용자 해시 체인에 대한 독점 제어권 획득.
4
읍면동 노드 서명 기반 잠금 관리 — 사용자 해시 체인은 읍면동 노드의 디지털 서명으로만 잠금·해제 가능. 임의 변경 원천 차단.
5
상위 계층 실시간 모니터링 — 읍면동 노드는 시군구가, 시군구 노드는 광역이 실시간 감시. 비정상 활동 즉시 상위 계층 대응.

핵심 프로토콜

해시 체인 기반 계층 간 거래 검증 프로토콜 — 5단계

거래 당사자와 최하위 공통 계층 노드 간에 해시 체인을 연동하는 핵심 프로토콜입니다. 하나의 거래가 변조되면 연동된 모든 체인에서 불일치가 즉시 발생합니다.

1

거래 데이터 해시 생성

당사자 A와 B 각각이 동일한 거래 데이터를 독립적으로 해싱하여 거래 해시를 생성합니다. 각 해시에 본인의 디지털 서명을 첨부합니다.

H_A = SHA-256(거래 데이터) + 서명_A  |  H_B = SHA-256(거래 데이터) + 서명_B
2

최하위 공통 계층 노드 전송

시스템이 두 당사자의 계층 정보를 비교하여 최하위 공통 계층을 탐색합니다. A와 B 각각이 자신의 거래 해시와 서명을 해당 노드로 전송합니다.

3

노드 서명 검증 + 해시 일치 확인

공통 계층 노드가 양측 서명을 검증하고 H_A = H_B 일치 여부를 확인합니다. 불일치 시 거래 거부 통지를 양 당사자에게 전송합니다.

4

노드 해시 체인 갱신 + 노드 해시 생성

노드는 자신의 기존 최종 해시와 거래 해시를 결합하여 새 노드 해시(H_N)를 생성하고, 이를 양 당사자에게 전송합니다.

H_N = SHA-256(H_NodePrev ‖ H_A)
5

당사자별 해시 체인 갱신

A와 B 각각이 H_N을 수신하여 자신의 해시 체인을 갱신합니다. 이로써 세 체인(A·B·노드)이 동일한 H_N을 공유하는 삼각 연동 구조가 완성됩니다.

H_A' = SHA-256(H_APrev ‖ H_N)  |  H_B' = SHA-256(H_BPrev ‖ H_N)

def classify_content(content: str) -> dict:
    # 콘텐츠에서 키워드 추출 및 서랍별 점수 산출
    keywords = extract_keywords(content)
    scores = {'금융': 0, '의료': 0, '교육': 0,
              '행정': 0, '교통': 0, '일반': 0}

    for kw in keywords:
        for drawer, kw_list in DRAWER_KEYWORDS.items():
            if kw in kw_list:
                scores[drawer] += kw_list[kw]  # 가중치 누적

    best = max(scores, key=scores.get)
    total = sum(scores.values())

    # 신뢰도 = (최고 점수 / 총 점수) × 100 + 최고 점수
    confidence = (scores[best] / total * 100 + scores[best]) if total > 0 else 0

    # 동점 시 일반 서랍 선택
    if list(scores.values()).count(scores[best]) > 1:
        best = '일반'

    return {'drawer': best, 'confidence': confidence, 'scores': scores}
